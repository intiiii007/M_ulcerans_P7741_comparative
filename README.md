# *M. ulcerans* P7741: Draft assembly and comparative gene‑content analysis

A two‑genome comparison between *Mycobacterium ulcerans* P7741 and the reference strain Agy99.

## Key results
- De novo assembly (SPAdes, 496 contigs)
- 25,176 SNPs/indels vs. Agy99 (99.14% identity)
- Gene‑content comparison (Roary): **4,163 core** gene families, **2,341 shell** families
